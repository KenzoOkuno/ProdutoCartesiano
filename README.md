# Simulador Interativo de Produto Cartesiano

Este projeto é um programa em Python que permite gerar e visualizar o produto cartesiano entre dois conjuntos definidos pelo usuário. A interface gráfica facilita o estudo e compreensão do conceito matemático de produto cartesiano.

## Funcionalidades

- Entrada interativa dos conjuntos \(A\) e \(B\) pelo usuário.
- Geração automática de todas as combinações do produto cartesiano \(A \times B\).
- Exibição dos pares ordenados resultantes em uma lista.
- Cálculo e exibição da cardinalidade dos conjuntos e do produto cartesiano.
- Atualização dinâmica conforme o usuário altera os conjuntos.


### Pré requisitos

- Python 3.x
- Biblioteca Tkinter (normalmente incluída na instalação padrão do Python)

### Instalação

Não é necessária instalação adicional. Basta ter Python com Tkinter disponível.

### Passo a passo

1. Execute o programa Python.
2. No campo "Conjunto A", digite os elementos do conjunto \(A\), separados por vírgulas.
3. No campo "Conjunto B", digite os elementos do conjunto \(B\), separados por vírgulas.
4. Clique no botão "Gerar Produto Cartesiano".
5. Visualize os pares ordenados do produto cartesiano na lista abaixo.
6. Confira as informações sobre a cardinalidade dos conjuntos.
